# 🌾 AI Crop Recommendation System

An intelligent crop recommendation platform designed to help farmers make data-driven decisions based on soil and environmental conditions.

This project combines modern web technologies with machine learning to suggest the most suitable crops for given inputs such as soil nutrients, temperature, humidity, and rainfall.

---

## 🚀 Tech Stack

**Frontend**

* Next.js (React)
* Clean, responsive UI

**Backend**

* FastAPI (Python)
* ML model integration

**Database & Auth**

* Supabase (PostgreSQL + Authentication)

---

## 🏗️ Project Structure

```
crsapp/
├── crs-client     # Frontend (Next.js)
└── crs-serverpy   # Backend (FastAPI)
```

---

## ⚙️ Features

* User authentication (Supabase Auth)
* Soil data input system
* AI-based crop recommendation
* Scalable backend architecture
* Clean monorepo setup

---

## 🔄 Development Workflow

* `main` → stable production branch
* `dev` → active development
* `feature/*` → new features

Each feature is developed in isolation and merged into `dev`, then promoted to `main` after stability.

---

## 🌐 Deployment

* Frontend → Vercel
* Backend → Render
* Database → Supabase
